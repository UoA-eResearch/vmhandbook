## Restore files from Backup

Research-drives and vms are backed up nightly by Connect, and you can restore files from backup yourself.

## Windows

1. Open the backup archive client by searching for the string "bac" in the search box.

[Search for backup archive client]('/backup/bacarchive_client.png')

2. Select Restore from the Backup options

[Select restore ]('/backup/select_backup_and_restore.png')


3. Select Drive as Source

[Select restore ]('/backup/select_drive_as_source.png')
